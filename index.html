California Enrollment Changes

The purpose of this project was to summarize changes in enrollment across the state of California.

Our goal was to determine whether unemployment rates across the state of California were correlated with public school enrollment changes. The initial hypothesis was that unemployment rates had a strong inverse correlation with public school enrollment in California. Ultimately, we found that unemployment and public school enrollment had little to no correlation. However, public school enrollment has significantly decreased since 2015 in the state of California, so more analysis is needed to determine the root cause.

To successfully analyze these trends and relationships, we utilized the U.S. Census Bureau API service and downloaded files from the California Department of Education. Our analysis was conducted using the Pandas and matplotlib libraries.
